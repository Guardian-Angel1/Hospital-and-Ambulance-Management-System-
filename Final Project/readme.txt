How to run the IITG Hospital & Ambulance Management System using PHP and MySQL

1. Download the project zip file

2. Extract the file and copy final folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name final

6. Copy the datebase with name final.sql and copy it and put in SQL Query part to make the database

7. Run the script http://localhost/Final Project
